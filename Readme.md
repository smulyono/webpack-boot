# Webpack-boot Project

Multi package project using [lerna](https://lernajs.io/). 

## Getting started

```bash
$ npm install -g yarn lerna
```

## Current Packages

* Yeoman Generator for webpack-boot

* Webpack boot project

## Starting development
```bash
$ yarn install
$ lerna bootstrap
```

## Publish changes

```bash
$ yarn install 
$ yarn deploy
```
